# 从 IntelliJ IDEA 说起

IntelliJ IDEA 是一种商业化销售的Javaopen in new window集成开发环境open in new window（Integrated Development Environment，IDE）工具软件open in new window，由JetBrainsopen in new window软件公司（前称为 IntelliJ）开发，提供Apache 2.0open in new window开放式授权的社区版本以及专有软件open in new window的商业版本，开发者可选择其所需来下载使用。

最初版于 2001 年 1 月时推出，当时是少数使用前阶代码浏览及代码重构open in new window的 Java 集成开发环境之一。

记得还是在 15 年的时候，那个时候 Eclipse 和 MyEclipse 还是如日中天。

印象中那个时候最火就是 MyEclipse 了，很多组件都是集成好的, 不过也是需要收费的。

而 Eclipse 是开源免费的，但是插件少，需要自己安装。


在那个年代 MyEclipse 是当之无愧的大哥

我自己第一次使用 IntelliJ IDEA 还是在学习 Scala 的时候，刚开始我是非常排斥的，不过没办法那个 时候好像就只有「IntelliJ IDEA」有插件支持，确实没办法，那个时候心里想，这软件配置是真的麻烦啊，我自己想要打一个 Jar 包，感觉要比 Eclipse 复杂一万倍。

不过那个时候也是能感觉到 大家慢慢的在往 **IntelliJ IDEA **上面转，后面就有意的使用 IntelliJ IDEA。
