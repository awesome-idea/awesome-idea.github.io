# 创建 Project

在 IntelliJ IDEA 中，一个项目可以帮助您将源代码、测试、使用的库、构建说明和个人设置组织到一个单元中
启动 IntelliJ IDEA。如果打开欢迎屏幕，请单击 New Project 。否则，从主菜单中选择 File | New | Project。

在 New Project 向导中，从左侧的列表中选择 Java。
